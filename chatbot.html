ChatBot 프로젝트 상세

자연스러운 대화와 확장성을 갖춘 챗봇 시스템

사용자와 자연스럽게 대화할 수 있는 챗봇 프로젝트입니다. 다양한 백엔드 기술을 연동하여 확장 가능한 구조로 설계되었습니다.

주요 기능

  • Spring Boot와 FastAPI를 이용한 하이브리드 백엔드
  • MongoDB를 사용한 대화 로그 저장
  • React 기반의 실시간 프론트엔드

기술 스택

Spring Boot, FastAPI, MongoDB, React

프로젝트 개요

TreeNut AI ChatBot은 TreeNut 팀이 직접 모델 학습을 시켜 캐릭터 챗을 서비스하는 대화형 인공지능 LLM 서비스입니다.

시연 영상

👊 사용 기술

☕ Spring Boot
유저, 채팅기록 불러내기 등 백엔드 서버에 사용
⚡ FastAPI
AI 서버와의 질문, 답변을 전달하기 위해 사용
🐬 MySQL
유저 정보, MongoDB 채팅 기록 ID 저장에 사용
🐳 Docker
컨테이너 기반의 배포 및 환경 관리에 사용
⚛️ React
ChatBot 서비스, 관리 프론트 페이지 제작에 사용
🌱 Node.js
ChatBot 데이터 관리 전용 서버에서 사용
🌿 MongoDB
채팅 기록 저장을 위해 사용
☁️ Google Cloud Platform
Google OAuth2 소셜로그인, Google Drive를 통한 캐릭터 이미지 저장에 사용
